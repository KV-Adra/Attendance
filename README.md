# 📚 Attendance Generator

A simple and efficient web application designed to help teachers quickly generate the daily absentee list of their class.

## 🌟 Features

- 📋 Roll number-wise student list
- 🔍 Search students by name or roll number
- ☑️ Mark absent students using checkboxes
- 🎨 Highlight absent students in red
- 👦 Boys Present counter
- 👧 Girls Present counter
- 👥 Total Strength, Present & Absent counters
- 📅 Automatically adds today's date
- 📄 Generates a formatted absentee list
- 📋 One-click copy to clipboard
- 📱 Responsive design for desktop and mobile

## 🎯 Purpose

This project was developed to simplify the daily attendance process in schools. Instead of typing absentee names manually every day, teachers can simply select absent students, generate the list, and copy it instantly.
